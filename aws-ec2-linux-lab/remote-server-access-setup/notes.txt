# AWS EC2 Remote Access - Notes

- EC2 (Elastic Compute Cloud) provides virtual Linux servers in the cloud.

- A key pair (.pem file) is required for secure authentication instead of passwords.

- MobaXterm is a GUI-based SSH client that simplifies remote access to Linux servers.

- SSH (Secure Shell) uses port 22 for encrypted communication between client and server.

- Security Groups act as firewalls and must allow inbound SSH (port 22) from the user's IP.

- The default username for Ubuntu EC2 instances is "ubuntu".

- Public IP address is used to connect to the instance from the internet.

- File permission for the .pem key must be restricted (read-only) to avoid security issues.

- Basic Linux commands help verify successful login and system status.

- Always close the session properly after use to maintain security.
